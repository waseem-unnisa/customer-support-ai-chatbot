# AI-Based Customer Support Chatbot with Query Analysis

## Project Description
This project focuses on automating customer support using a Python-based chatbot.
The chatbot interacts with users, answers common queries, and stores user questions
for analysis. The collected data is analyzed to identify the most common customer
issues such as refunds, returns, order status, and cancellations.

## Features
- Automated chatbot responses for common customer queries
- Stores user queries in CSV file
- Analyzes customer queries to find most common issues
- Basic Natural Language Processing using keyword detection

## Technologies Used
- Python
- Pandas
- CSV
- Basic NLP (Keyword Matching)

## Research Objective
The objective of this project is to analyze customer queries and develop an automated
response system to reduce human workload and improve customer support efficiency.

## Future Improvements
- Implement Machine Learning for automatic response prediction
- Add sentiment analysis for customer queries
